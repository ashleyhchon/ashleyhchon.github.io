AI Song Analysis

Python  |  Sep - Oct 2020

‍

For this project, I built a program that takes in lyrics for a song and determines its genre. Rather than hard-coding a set of rules, I used machine learning to classify songs.

I was provided with a set of training data: song lyrics with known genres. When I wanted to classify a new song into its genre, I found the most similar song in the training data and returned its genre.

In order to find the most similar song in the training data, I calculated term-frequency-inverse document frequency (tf-idf) weights for every song. These weights are essentially word counts, modified by how common various words are. I calculated the same weights for the input song. Then, I compared the input song’s weights to each song in the training data using cosine similarity, a measure of the similarity of two sets of weights.

In this project, I worked with data sets of varying sizes, got an introduction to machine learning, and practiced implementing written algorithms. I was also introduced to cultural considerations in universal design, and reflected on the global effects of A.I. systems.
