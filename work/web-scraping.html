Craigslist Web Scraping

For this project, I scraped some data from Craigslist and wrote a few queries over my scraped data. Specifically, I examined rental housing listings for a number of major US cities, plus Providence.

My scraper was a function called scrape_data. It took in a dictionary where the keys were city names from the CITIES list and the values were BeautifulSoup objects. My scraper used BeautifulSoup methods to transform the data into a format that I could query in my query functions.

I scraped the following data from each rental listing:

  • Price
  • Number of bedrooms
  • Description (i.e., the text of the link to that listing)

I also wrote several queries over my data:

  • A function to find the average number of bedrooms across all of the cities. The function took in the data and returned a float representing the average bedrooms.
  • A function to find the city with the highest average price for a given number of bedrooms. The function took in the structured data and a number of bedrooms and returned a city name.
  • A function to find the most commonly-occuring “interesting” word in the listing for a given city. The function  used the interesting_word function in the starter code to determine which words were interesting, and counted upper- and lower-case versions of a word as being the same. It took in the data and a city name and returned a single word.

Indeed Web Scraping

In the second iteration of the project, I scraped Indeed, a website often used by college students to find internship listings. I implemented a scraper that took in a concentration name as a string and outputted a CSV file that displayed all of the listings in a more cohesive way. The CSV file had values for the job title, company name, location, salary, position summary, date posted, date that the data was scraped, and a link to the actual Indeed listing.

I also wrote query functions that would find the company and location with the most listings for a given concentration. The last function I implemented found the concentration that had the most internship listings.
